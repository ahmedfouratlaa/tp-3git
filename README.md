# Mon Premier Journal Git

## Description
Exercice pratique pour apprendre Git et GitHub avec VS Code.

## Etapes réalisées
1. Configuration de Git (user.name, user.email)
2. Initialisation du dépôt (git init)
3. Création du fichier journal.txt et staging (git add)
4. Premier commit
5. Modification et comparaison (git diff)
6. Retour en arrière (git restore)

## Commandes utilisées
- git init
- git add
- git commit
- git diff
- git restore
- git push
